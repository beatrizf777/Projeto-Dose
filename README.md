      # D.O.S.E - Dashboard de Operações e Sistema de Estoque

O objetivo deste projeto é desenvolver um sistema de gestão de estoque voltado para pequenas farmácias, oferecendo uma solução simples e confiável para o controle de entradas e saídas de produtos, acompanhamento de itens mais vendidos e suporte na tomada de decisão para reposição junto a fornecedores. Com isso, busca-se reduzir as divergências entre o estoque físico e o registrado no sistema, minimizar erros operacionais e oferecer aos gestores e colaboradores uma ferramenta acessível que torne o processo de gestão mais eficiente, organizado e baseado em dados reais.
